# Ammar Malick Portfolio

Interactive personal portfolio built with HTML, CSS, and vanilla JavaScript.

## Highlights

- Smooth motion system (cursor follower, parallax, reveal-on-scroll)
- Glassmorphism UI with neon accents and responsive layout
- Experience timeline with expandable role details
- Skills stack with tool/language badges
- GitHub project showcase cards

## Tech Stack

- `HTML5`
- `CSS3`
- `JavaScript (vanilla)`

## Run Locally

Open directly:

- Double-click `index.html`

Or run a local server:

```bash
python3 -m http.server 5173
```

Then visit:

- `http://127.0.0.1:5173`

## Structure

- `index.html` - page structure/content
- `styles.css` - visual system, layout, animations
- `script.js` - interactivity and motion behaviors
- `assets/` - profile/visual assets
- `favicon.svg` - browser tab icon
